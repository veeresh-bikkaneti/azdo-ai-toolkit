# Contributing to AI Initiative Artifacts

> [!WARNING]
> **READ-ONLY ARTIFACT**
> This repository is designed as a stable reference implementation ("Golden Artifact").
> It is intended to be used as a **template** or **reference** for other projects.

## How to "Play With" This Artifact

### Option 1: Use as Template (Recommended)
If this repository is set as a **Template Repository**:
1. Click the green **Use this template** button at the top of the repository page.
2. Select **Create a new repository**.
3. This creates a fresh copy in your account with a clean git history, ready for your experiments.

### Option 2: Fork and Clone
1. Click the **Fork** button in the top right.
2. Clone your fork locally:
   ```bash
   git clone https://github.com/YOUR-USERNAME/Prompt-Manager.git
   ```

## Contribution Policy

To maintain the integrity of this reference artifact:
*   **Direct Pushes**: Disabled (Repo is effectively "Locked").
*   **Pull Requests**: May be restricted. Please verify functionality in your own fork/template copy.
*   **Issues**: Open an issue if you find a critical bug in the reference implementation.
